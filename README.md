# Driver UI Vehicle System

Vehicle system project with GMSL deserializer integration.

## Project Structure

- `gmsl-deserializer/` - GMSL2 deserializer board design (from [antmicro/gmsl-deserializer](https://github.com/antmicro/gmsl-deserializer))

## GMSL Deserializer Integration

The GMSL deserializer project has been integrated into this repository. See [INTEGRATION_GUIDE.md](INTEGRATION_GUIDE.md) for details on how to use it in your KiCad project.

## Features

- GMSL2 to MIPI CSI-2 conversion
- Power over Coax (PoC) support
- 50-pin Antmicro Dual Camera interface

## Setup

1. Clone this repository
2. Open KiCad project files
3. Reference GMSL deserializer schematics as needed
